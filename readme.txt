Exercice -> Mise en place d'un site de gestion de salle de sport
Pour la partie admin : nom et mot de passe : admin

liste des fichiers:

index.php:
    page d'acceuil avec fenêtre de connection et création de compte.

new_account.php:
    page de creation de compte utilisateur.

client.php:
    page de selection d'activité et de durée, qui permettra de calculer la depense d'energie effectué

add_client.php:
    script qui permet de rajouter un client dans la base de donnée

preview_validate.php:
    page de confirmation du choix et de la durée d'activité ainsi que de la dépense d'energie pour cette session

line_chart.php:
    affiche le recapitulatif imc et d'activité du client    

add_activities.php:
    formulaire qui permet de rajouter des activitées // partie Admin

database.php:
    fichier de connection à la base de donnée

graph_history_process.php:
    script qui permet de rajouter dans la bdd les informations de la session d'activité dans l'historique d'activité du client
